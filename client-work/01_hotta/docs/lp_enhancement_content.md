# えりさんの LP 強化コンテンツ案

## セクション 1: なぜ WordPress を使わないのか

### 見出し

**あえて WordPress を使わない、3 つの決断**

### 本文

多くの制作会社が採用する手法をあえて捨て、**独自のコード記述 × クラウド直結**という最新手法にこだわっています。

#### ① 企業の「資産」を守る徹底したセキュリティ

従来のサイト制作(WordPress 等)は、システムが常にネット上で動き続けているため、常に「攻撃の隙」があります。私の手法は、システムを動かさず「完成したデータ」だけを直接配信するため、改ざんや乗っ取りのリスクを極限まで排除しています。

#### ② 「表示 0 秒」を目指す圧倒的なスピード

ページを開くたびに重いプログラムが動く従来の方法とは違い、あらかじめ最適化されたコードを直接読み込ませます。特にスマホで見る求職者の「待てない」ストレスをゼロにし、チャンスを逃さないサイトを実現します。

#### ③ 固定費という「無駄な支出」をカット

サーバー(場所代)を借りる必要がない特殊なクラウド環境を利用します。これにより、「何もしなくても毎月数千円〜数万円かかる」という業界の当たり前を覆し、**維持費 0 円**を可能にしました。

## セクション 2: 公式 HP があっても採用 LP が必要な理由

### 見出し

**「立派なホームページがあるのに、なぜ LP が必要?」**

### 本文

#### HP と LP は、役割が違います

| 項目     | 公式 HP                    | 採用 LP                  |
| -------- | -------------------------- | ------------------------ |
| **目的** | 企業の信頼性を担保する     | 応募への意欲を最大化する |
| **対象** | 取引先、金融機関、顧客など | 求職者のみ               |
| **情報** | 網羅的(情報の広さ)         | 特化的(情報の深さ)       |

#### 「スペック」を「ストーリー」に翻訳する

HP の「創業 70 年」「官公庁工事実績多数」といった硬い事実を、LP では「どんな喜びがあるのか」「日々の現場でどう体現されているのか」という物語へ変換します。

#### 広告運用と採用コストの最適化

求人媒体や SNS 広告から HP のトップページにリンクさせると、ユーザーは他のページに目移りして離脱してしまいます。施工管理職の採用なら、施工管理の魅力だけを語る専用 LP へ着地させることで、情報のノイズを排除します。

## セクション 3: サーバー代 0 円の仕組み(既存セクションに統合)

### 「なぜ月額管理費を売らないのか」セクションに追加する内容

#### 技術的な裏付け

従来のような「場所を借りる(サーバー)」という仕組みではなく、最新のクラウド技術を活用し、データを直接配信する仕組みです。

**従来型:**

- サーバーという「場所」を借りる
- 月々数千円〜数万円の固定費

**私の手法:**

- クラウド直結型インフラ
- 場所代としての固定費を徹底排除
- **維持費 0 円**

これは「無料だからいい」のではなく、**「最高に安全で速い手法を選んだら、結果として維持費も無料にできた」**という順序です。

## 比較表(提案資料用)

| 比較項目         | 一般的な WordPress           | 私の提供する構成               |
| ---------------- | ---------------------------- | ------------------------------ |
| **表示速度**     | 標準(プラグイン等で遅くなる) | 極速(離脱を最小限に抑える)     |
| **セキュリティ** | 脆弱性があり、対策が必須     | 極めて高い(攻撃対象にならない) |
| **維持コスト**   | サーバー代+保守費用が必要    | サーバー代 0 円 + 保守不要     |
| **資産性**       | 数年でシステムが古くなる     | コードとして半永久的に残る     |
