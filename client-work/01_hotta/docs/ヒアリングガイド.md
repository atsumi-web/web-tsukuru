# 【堀田建設様向け】社員ヒアリング実施ガイド（採用 LP 制作）

今回のヒアリングは、単なる社員紹介ではなく、**「求職者の不安を解消し、自社のファンにする」**ための戦略的なコンテンツ作りを目的としています。

このガイドは、ヒアリングの意図を担当者様や対象者様に共有するためにご活用ください。

## 1. 今回のヒアリングの目的

採用 LP を訪れる求職者は、「自分にできるだろうか？」「自分に合った場所だろうか？」という強い不安を持っています。
社員様の**「等身大の言葉」**を通じて、以下の 3 つの価値を伝えます。

- **安心感**: 未経験や若手でも、失敗を許容し育てる文化がある。
- **納得感**: なぜ大手ではなく「地元密着の堀田建設」なのか。
- **期待感**: ここで働くことで、どのようなキャリア（成長）が待っているか。

## 3. ヒアリング時のポイント

### 「失敗談」や「迷い」こそが最大の武器

求職者が最も共感するのは、成功体験よりも「苦労した話」や「無理だと思った瞬間」です。本音を引き出せると、LP の信頼性が格段に上がります。

### 「具体的な一言」を拾う

「その時、具体的になんと言われましたか？」と聞いてみてください。その具体的なセリフが、LP の強力なキャッチコピーになります。

## 4. 相手に合わせた資料の渡し方（アドバイス）

### 堀田建設の担当者様（窓口の方）へ

**【全内容（フルセット）を渡す】**
質問項目だけでなく、「設計思想」もすべてお見せして良いと思います。えり様のプロとしての戦略が伝わります。

### インタビューを受ける社員様（本人）へ

**【「質問項目」と「はじめに」のみを渡す】**
裏側の狙い（編集メモなど）は伏せておくのがベストです。ありのままの本音を話していただくことが最も重要だからです。
