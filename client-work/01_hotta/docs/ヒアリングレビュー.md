# 社員ヒアリングレビューと LP 活用アドバイス

えりさんに作成いただいたヒアリングシートに対する、LP 制作の観点からのアドバイスです。

## 1. 菅原さん（異業種からの転職）

- **役割**: 「未経験でも大丈夫」という安心感の象徴。
- **ポイント**: Q3（無理だと言えた瞬間）を活用し、会社の教育体制の温かさを強調しましょう。

## 2. 岡安さん（地元密着・安定）

- **役割**: 「なぜ大手ではなくここなのか」という納得感の提供。
- **ポイント**: Q7（大手との比較）での回答を、LP の重要な見出しとして使いましょう。

## 3. 渥美さん（新卒・若手）

- **役割**: 「若さや学歴に関係なく成長できる」という希望の証明。
- **ポイント**: 21 歳での建築士取得と「未完成でいい」という言葉の対比を活かしましょう。

追加で「その時、先輩に具体的になんと言われましたか？」と深掘りすることで、LP で一番目立たせることができる「キラーワード」が見つかります。
