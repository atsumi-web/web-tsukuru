```
「私たちが、続けている理由。」(現状維持)
```

```
「鬼。悪魔。でも時々、神。」
「最高のキャプテンに、なれ。」

```

```
「離職率0%。その理由は、ここにある。」
「成長できる。続けられる。証明された環境。」
```

#### Pattern E: 地域貢献型

```
「100年後の街に、君の誇りを残す。」
「『あした』の亀山を、つなぐ仕事。」
```

## 📐 サイト構成の分析

### 共通する優れた構成パターン

#### パターン 1:ストーリー重視型(e-TEC、TWOSTONE)

```
1. 強烈なキャッチコピー
2. ストーリー(社員の体験談)
3. 仕事内容
4. 働く環境
5. 募集要項
```

**特徴**: 感情に訴えかけ、共感を生む
**堀田建設への適用**: Pattern B, E に最適

#### パターン 2:データ重視型(黒部エムテック)

```
1. 会社紹介
2. トップメッセージ
3. データで見る会社
4. スタッフインタビュー
5. 募集要項
```

**特徴**: 客観的、安心感を与える
**堀田建設への適用**: Pattern D に最適

#### パターン 3:成長重視型(グローウィン)

```
1. 哲学的キャッチコピー
2. 成長インタビュー
3. 卒業生インタビュー(キャリアパス)
4. 成長制度・研修
5. 募集要項
```

**特徴**: キャリア志向の強い人材に訴求
**堀田建設への適用**: Pattern B に最適
