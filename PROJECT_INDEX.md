```
01_hotta/
├── lp/main/           # メインLP
├── hub/              # リソースハブ（全資料へのリンク集）
├── command_center/   # 採用司令室（クライアント向けダッシュボード）
├── interviews/       # 社員インタビュー資料
└── docs/            # プロジェクトドキュメント
```

主要ファイル:

- `lp/main/index.html` - 採用LP本体
