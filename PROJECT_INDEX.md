## 📁 プロジェクト構造

### `site/` - 公開サイトディレクトリ

このフォルダ内は3つのゾーンに分かれています:

#### 🌐 **公開サイト本体** (site/直下)

web-tsukuru.jp の実際の公開ファイル

- `index.html` - トップページ(ポートフォリオ)
- `service.html` - サービス紹介ページ
- `works.html` - 制作実績一覧ページ
- `blog/` - ブログ記事(21記事)
- `css/`, `js/`, `images/` - サイトアセット

#### 📦 **site/client-work/** - クライアント納品物

完成したクライアント案件の納品物置き場

```
client-work/
└── 01_hotta/         # 堀田建設株式会社 採用LPプロジェクト
    ├── lp/main/      # メインLP (将来的にドメイン割当用)
    ├── hub/          # リソースハブ
    ├── command_center/ # 採用司令室
    ├── interviews/   # 社員インタビュー資料
    └── docs/         # プロジェクトドキュメント
```

#### 🛠️ **site/works/** - 作業部屋

開発中・テスト中のファイル、実験的な資料

- 現在は堀田建設案件のURL変更できないファイルを一時的に保管中
- 将来的には開発中テンプレートやテストページの置き場所

#### 📚 **site/templates/** - テンプレート集

再利用可能なテンプレート・コンポーネント(241ファイル)

#### 📖 **site/manuals/** - 公開マニュアル

クライアント向け公開マニュアル

- `client/` - クライアント向けマニュアル(15ファイル)
  - アカウント準備、LINE、GA4、求人サイトなど

---

### ルート直下のフォルダ

#### `wireframe_components/`

- ワイヤーフレーム用コンポーネント(9ファイル)）

#### `manuals/`

内部管理用マニュアルハブ(統合インデックス)

マニュアル構成:

- `index.html` - マニュアル総合インデックス
- `admin/` - 管理者向け(10ファイル)
- `internal/` - 内部向け(25ファイル) - コーディング規則など
- `daily/` - 日次作業用(7ファイル)
- `marketing/` - マーケティング資料
- **注**: クライアント向け公開マニュアルは`site/manuals/client/`に配置

#### `planning/`

プロジェクト計画関連

主要ファイル:

- `blog_ideas.md` - ブログ記事ネタリスト
- その他企画資料

### 🔧 開発環境

#### `.vscode/`

- `settings.json` - UTF-8エンコーディング強制設定

#### `.editorconfig`

- エディタ共通設定

#### `scripts/`

- 開発用スクリプト（12ファイル）
