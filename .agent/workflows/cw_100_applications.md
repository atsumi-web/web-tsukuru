# CW 案件応募ワークフロー Ver2.0（高単価・プロトタイプ先行型）

> **アップデート（2026/04）**: 「とにかく100件送る（量の勝負）」のフェーズは終了。幼稚園サイト実装の経験を生かし、これからは**「高品質な自作デモを見せ、あなた用のトップページも無料で作りますよ」とオファーする『先行プロトタイプ戦略』**で、30万〜50万円以上の高単価案件をもぎ取る【勝率特化型】のワークフローにシフトする。

---

## 🎯 応募戦略の2つの軸

### 🔥 軸1：プロトタイプ先行提案（本命・高単価）
- **対象**: 予算30万円〜50万円以上のLP・コーポレートサイト案件。
- **武器**: みどりが丘幼稚園のデモサイト（品質証明）＋「2〜3日で無料試作」の最強オファー。
- **効果**: 過去の実績比べ（コンペ）を無力化し、「自分たちのために先に作ってくれる」という手厚さで即決受注を狙う。

### ✅ 軸2：通常応募（サブ）
- **対象**: 予算数万円〜10万円程度のバナー、チラシ、小規模な改修など。
- **武器**: これまでのベーステンプレを活用し、Antigravityに任せてサクサク数をこなし、信頼と小銭を稼ぐ。

## STEP 1：案件を探す

### 優先カテゴリ（高単価LP・Web制作を第一に探す）
| 優先度 | カテゴリ | 狙いどころ・予算 |
|--------|---------|-----------|
| 🔥 激熱 | ホームページ制作・Webデザイン | フルスクラッチコーポレート・LP（30万〜50万円） |
| ✅ サブ | デザイン（全体） | チラシ・名刺・バナー・LINEリッチメニュー |
| ✅ サブ | 動画・映像 | Instagram Reels・AI動画生成 |

### ⚡ 絶対スルー基準（地雷案件：制作会社の下請けは絶対にやらない）
- **「当社はWeb制作会社です」等の下請け案件**（えりさんの「自由な提案・高速実装」が完全に殺され、マイクロマネジメントで疲弊するため）
- **テスト発注で金額を指定しているもの**（1,000円でのデザインテスト等。安くパーツを集めるだけの使い捨てリスク大）
- **AIデータ/PSD/Figma納品必須**（制作会社が後で自分たちでいじるための「作業員」扱いです。フルスクラッチ納品以外の案件は不可）
- WordPress必須（テーマやプラグインの依存は避ける）
- 複数アカウントで同文、20人以上の大量募集（使い捨て）
- 都内出社、既婚の有無等のプライバシー情報を要求
- 予算が安すぎる（最低ラインを下回る）

## STEP 2：応募文作成（Antigravityに丸投げ）

> 案件詳細をコピーして貼るだけで「応募すべきか判断」→「応募文カスタマイズ」まで依頼できる。

### 🤖 Antigravityへのプロンプト（コピペ用）

```
以下の案件を見てください。

1. この案件に「プロトタイプ先行提案」の軸で応募すべきか、または「通常応募」の軸で応募すべきか判断してください。地雷案件ならスキップしてください。
2. 応募する場合は、適切なテンプレをもとに、案件に合わせた応募文を作成してください。

【案件詳細】
（ここに案件ページの文章をそのままペースト）
```

## 💡 勝利への鉄則（Ver2.0）

1. **みどりが丘幼稚園を「実弾」として使い倒す**：あれがえりさんの証明書です。
2. **「過去の実績」より「今からあなたのために作ります」**：コンペや相見積もりを一撃で破壊するキラーワードです。
3. **AI（Antigravity）で即座にプロトタイプを作る**：2日でトップページを出せる武器があるからこそ出来る戦略です。自信を持ってオファーしてください。
4. **返信は24時間以内の即レス**：プロトタイプ提案に乗ってくるクライアントは「本気」です。スピード感で信頼を確定させます。
