# リール90本制作ワークフロー（確立版）

## ステップ1：テーマ設計

1. **テーマ数と構成を決める**（今回：15テーマ × 6本 = 90本）
2. **設計書 `reel_script_design.md` を作成**
   - テーマ名・ターゲット課題・6つの切り口を定義する
   - Safe Realismチェック：インボイス等の事実確認が必要なテーマは差し替える
   - テーマ間の重複確認（近似テーマは切り口で差別化）

## ステップ2：スクリプト執筆

3. **テーマ別CSVを1つずつ執筆**
   - 保存先：`planning/reels_v2/theme_XX_テーマ名.csv`
   - 列構成：`ファイル名, 表紙テキスト, フック, 問題, 解説, 解決, 最後1, 最後2`
   - 文字数制限：フック25字以内・問題/解説/解決各40字以内

## ステップ3：AIレビュー

4. **5テーマずつ他AIに確認依頼**（ChatGPT / Gemini / Claude / Perplexity など）
   - 確認プロンプト：
     ```
     以下のCSVをレビューしてください。
     ① 日本語の自然さ（不自然な語尾・文体の揺れ）
     ② Safe Realism（誇張・過保証・断言しすぎていないか）
     ③ 文字数：フック25字以内・問題/解説/解決40字以内
     問題があればその行と修正案を教えてください。
     [CSVの中身を貼る]
     ```
5. **フィードバックを即反映してCSVを上書き保存**
   - 「必ず」「ゼロになります」「9割」等の強い断言 → 弱める
   - 誤字・文字数超過 → 必須修正

## ステップ4：テンプレート別CSV自動生成

// turbo
6. **再配置スクリプトを実行する**
   ```
   node planning/reorganize_reels_v2.js
   ```
   - 15テーマ → A〜J（各9本）に自動振り分け
   - 出力先：`planning/reels_v2/canva/reel_A.csv` 〜 `reel_J.csv`

## ステップ5：Canvaで動画生成

7. **1日3〜4テンプレート（27〜36本）ずつDL**（上限対策）
   - CanvaでテンプレートAを開く → 一括作成 → `reel_A.csv` をインポート
   - 8列を対応付けて生成 → まとめてDL
   - テンプレートB〜Jで同様に繰り返す

- テーマ重複は執筆前の設計段階で潰しておくと後が楽
