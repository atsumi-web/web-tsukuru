# 🔗 ルール・ルーティング設定（AI専用）

当プロジェクトでは、コーディング規則やモバイル最適化ルールの二重管理を防ぐため、**「すべてのルールは特定のHTMLファイルに集約する（Single Source of Truth）」**という方針を採用しています。

AIアシスタント（Antigravity等）は、コーディング作業やレイアウト調整に着手する前に、**必ず以下のHTMLファイルを `view_file` ツールで直接読み込み**、そこに記載されているルール（特に「dvh禁止」「BEM・CSSの分け方」「文字化け対策」等）を100%厳守してください。

> ⚠️ **警告**
> このファイル（MD）自体には具体的なルールを追記しないこと。ルールは常に人間のダッシュボード（HTML）を中心に回る。
