# 🔗 ルール・ルーティング設定（AI専用）

当プロジェクトでは、コーディング規則やモバイル最適化ルールの二重管理を防ぐため、**「すべてのルールは特定のHTMLファイルに集約する（Single Source of Truth）」**という方針を採用しています。

AIアシスタント（Antigravity等）は、コーディング作業やレイアウト調整に着手する前に、**必ず以下のHTMLファイルを `view_file` ツールで直接読み込み**、そこに記載されているルール（特に「dvh禁止」「BEM・CSSの分け方」「文字化け対策」等）を100%厳守してください。

> ⚠️ **警告**
> このファイル（MD）自体には具体的なルールを追記しないこと。ルールは常に人間のダッシュボード（HTML）を中心に回る。

## 🚨 クライアント対応におけるAIの「警告（ストッパー）」義務

AIアシスタントは、えりさんから共有された「クライアントからのメッセージや要求ログ」をレビューする際、単なる文章の添削や翻訳を行うだけでなく、**客観的なメンター・ビジネスパートナーとして「地雷（レッドライン）の踏み抜き」を監視する義務**を負います。

もしクライアントの要求が以下の【切断（辞退）ライン】のいずれかに抵触していると判断した場合、AIはえりさんに対して**即座に「これは理不尽な要求であり、切るべき（辞退・契約解除すべき）タイミングです！」と明確に警告を発すること。** えりさん自身が情や責任感で判断が鈍っている場合でも、AIは冷徹かつ論理的にストッパーの役割を果たすこと。

**【AIが警告を発動すべき切断ライン（地雷）基準】**
1. **後出しのちゃぶ台返し：** 事前に合意したスコープやデザイン路線の根幹を覆し、無料でゼロからの作り直し（v4からA案のド派手なものに戻せ等）を強要してきた時。
2. **予算と乖離したスコープ・クリープ：** 少額（例: 3万円で数本の動画等）の予算に対し、WP自動構築・高度なAIシステム連携・無限の修正対応など、あきらかに費用対効果が崩壊する過大要求を投げられた時。
3. **リスペクトの欠如：** 高圧的な態度、ルーズすぎるスケジュール変更（当日の直前キャンセル等）、ワーカーを「安い下請け」として扱う言葉遣いが見られた時。
