# クラウドワークス応募用 テンプレート集

## 👑 1：【技術パートナー・コンサル向け】提案文
SNS運用会社やマーケティング会社で、Webの技術が足りていない「優良代理店」に対する、商談同席＆プロトタイプを武器にした提案文です。

```text
はじめまして。Web制作フリーランスの渥美絵里と申します。

「[募集にあった背景、例：住宅業界向けのHP制作における、商談同席から制作まで伴走できるパートナー]」という募集背景を拝見し、私の最も得意とする業界・立ち回りでしたので、真っ先に応募させていただきました。

現在、私は「[得意な業界、例：建設業・工務店・BtoB企業]」に特化したWebディレクションおよびフルスクラッチでの制作（デザイン〜コーディングまで）を主軸に行っております。

【制作実績・ポートフォリオ】
▼ 建設関連の制作例（堀田建設株式会社様 / 採用基盤構築）
https://web-tsukuru.jp/portfolio_cw.html

▼ 制作における品質・UIの参考デモ（みどりが丘幼稚園様）
https://web-tsukuru.jp/works_youchien.html
※スマホでの操作性と、各種コンバージョン（成果）を最大化する導線設計に特化しています。

【私が提供できる３つの価値】

① 商談時の「専門家」としてのサポート
提案の場に同席し、クライアントに「なぜこの導線が必要か」「技術的にどう構築するか」を専門用語を使わずに分かりやすくご説明いたします。

② 『保守・維持費ゼロ』の強力な提案構築
脆弱性や保守費用を嫌うクライアントに対し、最新の「静的サイト＋ヘッドレスCMS」を用いた【セキュリティが高く、かつ月々の保守費がかからない最適な構成】をご提案できます。相見積もりになった際の強力な武器になります。

③ 高速なプロトタイプ（試作）による受注率の底上げ（★ご提案）
もし現在、ご提案を控えている[ターゲット層、例：工務店様・住宅会社様]がごさいましたら、事前に参考サイト等のイメージをお聞かせください。
【2〜3日以内に、そのクライアント専用のトップページ（試作デザイン）を私が無料で作成】いたします。
商談時に実際のデモ画面をお見せできれば、圧倒的な確率で受注へ繋げることが可能です。

まずは「無料の試作品」で品質とスピードをご確認いただき、今後長くお力になれるかをご判断いただければ幸いです。
まずは一度、オンライン等でお気軽にお打ち合わせの機会をいただけないでしょうか？
ご連絡を心よりお待ちしております。

渥美 絵里
```

## 🥈 2：【直案件・B2B向け】プロトタイプ先行提案文
相手がB2B企業やエンド直案件の場合に、他社を出し抜いて即決させるための提案文です。

```text
はじめまして。Web制作フリーランスの渥美絵里と申します。

今回の「[案件タイトル、例：補助金申請支援のLP制作]」のご要望を拝見し、私が得意とするBtoBビジネス向けの「信頼感と誠実さが伝わるデザイン設計」に非常にマッチすると感じ、応募いたしました。

とはいえ、多数の応募文や過去の実績画像を見るだけでは、「実際のスマホでの操作感」や「本当に自社のイメージや商材に合うのか」をご判断いただくのは難しいかと存じます。

そこでご提案なのですが、もしよろしければ「ベンチマークにしている参考サイト」や「ご希望のカラー（青系、堅実なイメージ等）」を少しお伺いできないでしょうか？

ヒアリングさせていただけましたら、【2〜3日以内に、御社専用のトップページ（試作デザイン）を私が無料で作成】し、実物のデモをお見せいたします。

▼ 私が制作できる品質・UIの参考デモ（※先日3日で制作した別件のものです）
https://web-tsukuru.jp/works_youchien.html
※スマホでの操作性と、お問い合わせ・各種登録などのCV（成果）を最大化する導線設計に特化しています。

[商材名、例：補助金]という信頼性が第一となる商材ですので、まずはこのサンプルのようなクオリティで、御社専用のファーストビュー（試作）を構築いたします。

そのデモを実際のスマートフォン等で触って見ていただき、「これなら安心して任せられる」と確信していただけた場合のみ、正式にお任せいただくという流れでいかがでしょうか？（もちろん、イメージと違えば遠慮なくお断りいただいて構いません）

御社にとって完全にリスクゼロでご検討いただけますので、まずは「無料試作品」でお試しいただければ幸いです。

ぜひ一度、メッセージにてご要望のイメージをお聞かせください。
ご連絡を心よりお待ちしております。

渥美 絵里
ポートフォリオ：https://web-tsukuru.jp/portfolio_cw.html
```
