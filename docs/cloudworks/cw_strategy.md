# クラウドワークス応募戦略 Ver2.0（プロトタイプ先行型）

このドキュメントは、クラウドワークス上で「低単価な使い捨ての下請け」になることを防ぎ、高単価で自由な直案件（または優良な代理店・パートナー案件）を勝ち取るための戦略マニュアルです。

## 🎯 応募戦略の2つの軸

### 🔥 軸1：プロトタイプ先行提案（本命・高単価）
- **対象**: 予算30万円〜50万円以上のLP・コーポレートサイト案件（特に建設・B2B・法人向け）。
- **作戦**: 「みどりが丘幼稚園」のデモサイトを品質証明として提示し、「2〜3日で実際のデモ（プロトタイプ）を無料で作りますので見てください」とオファーする。
- **効果**: 過去の実績やプロフィールによる比較（コンペ）を無力化し、「自分たちのために先に動いてくれる」という手厚さとスピード感で即決受注を狙う。

### ✅ 軸2：通常応募（サブ）
- **対象**: 予算数万円〜10万円程度のバナー、チラシ、小規模な改修など。
- **作戦**: 既存の資産（テンプレート等）を活用し、AI（Antigravity）に任せてサクサク数をこなす。深入りせず、信頼と小銭を稼ぐ用途と割り切る。

## ⚡ 絶対スルー基準（地雷案件の見極め方）

以下の特徴が一つでも当てはまる案件は、えりさんの強み（提案・UI/UX設計・静的爆速CMS）が一切活きず、疲弊するだけなので**絶対に応募しない（即ブラウザバック）**こと。

1. **「当社はWeb制作会社です」等の完全な下請け案件**
   - 「ワイヤーフレームに基づいて」「サーバーはいじれないのでファイル納品で」等と明記されているものは、実働部隊を持たない中抜き代理店。
   - ただ色を塗るだけの作業員（駒）として扱われるためNG。
2. **システム・ツール縛りが厳しい案件**
   - 「WordPressでの構築必須」「Goope、Studioなどのテンプレートツール必須」など。保守費ゼロの静的構築モデルが使えず、単価が上がらない。
3. **異常な低単価と丸投げの矛盾がある案件**
   - 「デザインのみ10ページで5万円」「勝ちパターンはあると言うが素材は全てワーカーに用意させる」。
4. **出社・対面MTG必須案件**
   - 「週1の対面MTGあり」など。フリーランスの最大のメリットである「場所と時間の自由」が奪われ、実質的な社畜化する。
5. **詐欺・情報商材へのリスト集め案件**
   - 「初心者歓迎」「未経験OK」「お人柄重視です」「独立を視野に入れていますか？」などと書き、Zoomで面談をさせた上で高額スクールやコンサルを売りつける罠。

## ⭕ 狙うべき優良案件（ストライクゾーン）

1. **完全なエンドクライアント（事業主の直案件）**
   - 幼稚園、建設会社、士業、クリニックなど、自分たちでビジネスをやっており純粋にHPやLPを必要としている人たち。プロトタイプ提案が一番効く。
2. **良い代理店・コンサル（技術パートナー扱い）**
   - マーケティングやSNS運用の専門会社だが、Webの技術がないため、「商談から一緒に組める技術的なパートナー」を探している会社（GCLIP野中さんのパターン）。
   - クライアントの言いなりになるのではなく、専門家として裁量を持って構築提案ができるため非常に狙い目。

## 💡 勝利への鉄則

1. **みどりが丘幼稚園と堀田建設を「実弾」として使い倒す**。
2. **「過去の実績」より「今からあなたのために作ります」**で勝負する。相手にとってリスクゼロの最強オファー。
3. **AIを活用して即座にプロトタイプを作る**。2〜3日でトップページを出せる武器があるからこそ出来る戦略。
4. **返信は24時間以内の即レスを徹底する**。プロトタイプ提案に乗ってくるクライアントは「本気」なので、スピードで信頼を確定させる。
