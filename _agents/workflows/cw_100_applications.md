## ⚡ 一目でスルー確定パターン（Antigravityに聞く前にはじく）

| スキップ基準 | 理由 |
|------------|------|
| 複数アカウントで文章が一字一句同じ | 段階発注の悪用パターン |
| 募集人数20人以上 | 使い捨て案件の可能性が高い |
| 都内・出社あり | 三重県在住のため物理的に不可 |
| 年齢制限あり（20代・30代まで） | 応募資格なし |
| **コンペ × 3冠**（0件・本人確認未・発注ルール未） | 作品を集めて使い捨てのリスク大・絶対スキップ |
| **大量募集（20人以上） × 3冠** | 同上。試作集め前提パターン |
| 3冠のみ・固定報酬・内容が合致 | → まずメッセージで会話して判断（柔軟対応OK） |
| PSD・AIデータ納品必須 | Photoshop/Illustratorは不使用 |
| WordPress必須 | コードでの実装がえりさんの方向性 |
| 既婚の有無・住居環境を質問 | プライバシー侵害 |
| 🚨 テスト報酬に金額入力指定（460円・100円・50円など） | 既知の詐欺的パターン。絶対に応募しない |
| 🚨 無報酬トライアル（表紙1枚無料）×10人以上大量募集 | 試作の見せ集め前提 |
| 🚨 CWメッセージで顔写真・学校名・勤務先を要求 | プライバシー侵害 |
| 予算¥1,000以下 | 時間の無駄 |
| 発注者評価★3.0以下 | トラブルリスクあり |

> 💡 **応募数は気にしない**。AIで応募コストはゼロ。スキルが合っていれば100人でも送る。「当たればラッキー」精神でOK。

### ⚠️ SNS案件の注意（精査強化）
カフェ・ホテル・旅行系のSNS投稿案件は「LP受注ゴール」につながりにくい。  
以下に当てはまる場合はスキップ：
- 単価：1投稿1,500円以下
- SNS運用のみの小規模個人クライアント
- デザインだけでHP・LPの相談が起きにくい構造

→ **「HP・LP・チラシ」案件に集中することがゴールへの近道。**

## STEP 2 & 3：Antigravityに貼るだけ（判断 → 応募文）

> 案件詳細をコピーして貼るだけで「応募すべきか判断」→「応募文カスタマイズ」まで依頼できる。

### 🤖 Antigravityへのプロンプト（コピペ用）

```
以下の案件を見てください。

1. この案件に応募すべきか判断してください（YES/NO＋理由を簡単に）
2. YESの場合はベーステンプレをもとに、案件に合わせた応募文を作成してください

【案件詳細】
（ここに案件ページの文章をそのままペースト）
```

Antigravityが以下を返します：
- ✅/❌ **応募判断**（スキル合致・予算・勝算・クライアントリスク）
- 📝 **カスタマイズ済み応募文**（そのままコピペして送れる状態）

## 応募後の管理

セッション終了前に「今日見た案件を記録して」とAntigravityに依頼する。  
→ 評価記録（cw_log.html）への追記で次回以降の審査精度が上がる。

## 💡 鉄則5か条

1. **案件はAntigravityに判断させる**。自分で迷わない。貼るだけでOK
2. **ポートフォリオURLは毎回入れる**。無言の営業マン
3. **1日5件だけやる**。完璧より継続。毎日続ける方が大事
4. **返信は24時間以内に即レス**。スピードが信頼になる
5. **採用率1割でOK**。100件送れば10件取れる計算

> 「応募数が信頼を作る。最初の10件は実験、次の50件が学習、残り40件で仕事になる。」
