## 🌟 メインの仕事場（一番よく使う場所）
ここはえりさんが毎日バリバリ働く「表舞台」です。

### 🌐 `site/` （公開サイト・クライアントの部屋）
- **役割**: 世界中に公開される「本番のWebサイト」や、納品前の「クライアントの制作ルーム」がすべて入っています。
- **中身**: トップページ、`client-work/`（堀田様や他案件の部屋）、ダッシュボード（司令室）など。
- **危険度**: 🔴**絶対削除禁止**。ここを消すとお店が吹き飛びます。


### 🧠 `planning/` （社長の頭の中・企画室）
- **役割**: えりさんだけが見る、自社の経営戦略やブログのネタ帳です。
- **中身**: `business_strategy.md`（事業戦略）や `monitor_response_manual.html`（モニター応募時の即レス神マニュアル）など。
- **危険度**: 🟡**注意**。えりさんの大事な資産です。

### 🛠️ `scripts/` （便利な自動化工具箱）
- **役割**: めんどくさい作業を1秒で終わらせてくれる魔法の道具箱です。
- **中身**: `resize_images.ps1`（画像を自動でリサイズして軽くするプログラム）など。
- **危険度**: 🟡**注意**。

## 🤖 AIとシステムの裏側（触らないでOKな場所）
ここは「裏方の機械室」です。えりさんが直接ファイルを開いてイジることは滅多にありません。

### 🤖 `.agent/` （AIへの命令書）
- **役割**: 私（AntigravityなどのAI）が、えりさんの「dvh禁止」「文字化け対策」などの厳しいルールを読み書きするための「AI専用ルート」です。

### 📦 `node_modules/` （巨大な部品倉庫）
- **役割**: Tailwind CSSなどの便利なデザインツールを動かすための「裏側の部品」が何万個も詰まっています。（システムが勝手に用意しました）
- **危険度**: 🟢**削除OK**。（でも消すと一時的にCSSが使えなくなり、`npm install` という呪文で復活させる手間がかかります。基本は放置でOKです）

### 📝 Tailwind CSS（デザインツール）の「設定ファイル4兄弟」
ここにある4つのファイルは、上の巨大な部品倉庫（ツール）を動かすための**「発注書と設定書」**です。AIにデザイン改修を頼む際によく使われます。
- **`package.json`**（発注書）：「このサイトにはTailwind CSSが必要だ！」と書かれた指示書。🔴**絶対削除禁止**
- **`package-lock.json`**（レシート）：届いた部品のバージョン情報がぎっしり書かれた長いレシート。システムが勝手に作ります。🟡**削除注意**
- **`tailwind.config.js`**（カスタマイズ設定）：「ブランドカラーはこの青色」「余白はこのサイズ」など、専用のデザインルールを定義するファイル。🔴**削除注意**
- **`input.css`**（CSSのタネ）：Tailwindが最終的なCSSを作り出すための「ベース（素）」になるファイル。🔴**削除注意**

### 🛡️ `.vscode/` ＆ `.editorconfig` （文字化け防止の盾）
- **役割**: VS Codeに「絶対にUTF-8（文字化けしない魔法の文字コード）で保存しろ！」と強制するための魔除けのお札です。
- **危険度**: 🔴**削除禁止**。消すと、また「みらいへ、👉 譛ｪ譚･繧偵・」の悪夢がやってきます。

### ⏳ `.git/` ＆ `.gitignore` （タイムマシンの記録装置）
- **役割**: 過去の作業履歴をすべて記録し、間違えても「1段前に戻る！」ができるための透明な装置です。
- **危険度**: 🔴**絶対削除禁止**。消すと過去に戻れなくなります。

## ⚔️ 特別コラム：ルート直下にいる「7つの必須ファイル（7人の侍）」
えりさんが「フォルダの一番上（ルート）」を見たとき、ポツンと置かれている以下の7つのファイルは、プロジェクトを根底から支える**絶対に移動してはいけない裏方たち**です。
これらが綺麗に7つ並んでいる状態が、**「一番正しいプロの作業空間」**です！安心してそのままの場所に居させてあげてください。

1. **`README.md`**：えりさんのための「ガイドマップ看板」（今見ているこれ）
2. **`.editorconfig`**：VS Codeに文字化けを防がせるための「盾その１」
3. **`.gitignore`**：Gitに余計なものを記録させないための「盾その２」
4. **`package.json`**：デザインツール（Tailwind）の「発注書」
5. **`package-lock.json`**：デザインツールの「詳細なレシート」
6. **`tailwind.config.js`**：デザインツールの「カスタマイズ設定書」
7. **`input.css`**：デザインを作り出すための「タネ」
