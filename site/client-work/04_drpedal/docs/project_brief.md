## 02 目的とKPI

- **最優先目的**：修理工の採用応募獲得（目標：月2件）
- **第二目的**：法人・自治体向けサービスのリード獲得

## 03 ターゲット

### メインターゲット（採用）
| 項目 | 内容 |
|---|---|
| 年齢・性別 | 20〜30代・男性 |
| 居住地域 | 東京・福岡・茨城 |
| 職業・スキル | 自転車修理の経験者 |
| 行動起点 | 求人広告を見て、会社の信頼性を確認しに来る |
| 最大の不安 | 「まともな会社かどうか」（業界に不誠実な企業が多い） |

### セカンダリーターゲット（法人リード）
自治体・法人の担当者。採用訴求を優先しながら、サービスページへの明確な導線を確保。

## 09 導線設計

```
求職者フロー：TOP → Hero（興味喚起）→ About（共感）→ Strength/Clients（信頼）→ Recruit（応募）
法人フロー：TOP → サービスTOP → 各サービス詳細 → お問い合わせ
```

各ページ末尾に「採用情報を見る」「お問い合わせ」ボタンを常設。  
グローバルナビのヘッダーに採用ページへのリンクを目立つ形で配置。
