# 北海道カフェ案件 ミーティング準備シート

> 最終更新: 2026-03-26

## 1. ヒアリングシート（ミーティングで聞くこと）

### カフェの基本情報
- カフェの名前（決まっていれば）
- コンセプト・こだわりを一言で表すと？
- メインターゲット（地元の常連？観光客？どちらを重視？）
- よく参考にするカフェやお気に入りのサイトURLはあるか？

### ブランド素材
- ロゴはあるか？（なければ別途依頼or今回の範囲か）
- 使いたい色・雰囲気は決まっているか？
- SNSアカウントはあるか？（Instagramなど）→ サイトに埋め込みたいか？

### 機能・ページ
- お問い合わせは「電話のみ」か「フォーム」か？
- 予約機能は必要か？（Googleフォーム or 外部ツール）
- メニューは画像で貼り付けか、テキストで掲載か？
- Googleマップへのリンクで十分か？

### 技術・運用
- ドメインはあるか？（なければ取得サポートが必要）
- サーバーはあるか？（なければCloudflare Pagesで無料提案できる）
- 開店後の写真差し替えは自分でやりたいか？サポートを希望するか？

## 2. サイト構成案（5ページ）

| ページ | 内容 |
|---|---|
| TOP | ヒーロービジュアル・コンセプト一言・注目メニュー・お知らせ・アクセス地図（簡易） |
| 店舗について | オーナーの想い・コンセプト・店内の雰囲気（写真） |
| メニュー | ドリンク・フード・スイーツ（フリー素材→後に実写差し替え） |
| アクセス | 地図・住所・営業時間・定休日・電話番号・駐車場情報 |
| お問い合わせ | Googleフォーム連携（シンプルなお問い合わせフォーム） |

## 3. デザイン方向性

- **カラー**：北海道の自然をイメージ（森の深緑 + オフホワイト + 木材ベージュ + アクセントのテラコッタ）
- **フォント**：日本語は丸みのあるセリフ体（Noto Serif JP）+ 英字は細めのセリフ体
- **写真**：余白を広めに取り、写真を大きく見せる構成
- **全体感**：「日常を忘れてゆっくりできる場所」の非日常感を演出
