## ⑩ なぜ10万円なのか？

### 価格の根拠（論理的説明）

**比較対象：競合サービス・既存手段との差**

| 比較対象                     | 費用                    | えりさんとの差                                 |
| ---------------------------- | ----------------------- | ---------------------------------------------- |
| マイナビ・リクナビ等の掲載料 | 月3〜10万円（成果なし） | 月課金が続く「消耗品」。えりさんのLPは「資産」 |
| 一般Webデザイナーへの外注    | 30〜80万円              | 業界知識ゼロ。建設業のリアルを知らない         |
| 採用コンサルタント           | 50〜200万円             | 成果報酬型で高額。中小企業には重い             |
| 自社で制作（依頼なし）       | 数百時間の工数          | 採用に特化したノウハウなしで完成しない         |

**10万円の内訳イメージ**

| 作業                           | 市場相場        | 提供価格     |
| ------------------------------ | --------------- | ------------ |
| ペルソナ・競合分析（戦略設計） | 5〜10万円       | 含む         |
| LP制作（PC/SP）                | 30〜80万円      | 含む         |
| LINE導線設計                   | 5〜15万円       | 含む         |
| 採用司令室（ダッシュボード）   | 5〜10万円       | 含む         |
| 修正対応2回                    | 1〜3万円        | 含む         |
| **合計（市場相場）**           | **46〜118万円** | **¥100,000** |

**モニター価格としての位置づけ**

> 「通常価格は15〜25万円ですが、実績づくりのため現在はモニター価格10万円でご提供しています。  
> 建設・建築業に完全特化しているため、汎用デザイナーより圧倒的に速く、かつ採用に刺さる設計ができます。  
> 0円維持費の資産として、最低5年以上使えるLPを10万円で手に入れることができます。」
