# リール設計書 v3.0（次の90本）

## 概要

| 項目 | 内容 |
|---|---|
| 発信者 | Webとしくみをつくる（えりさん） |
| ターゲット | 建設・建築業の中小企業の社長（30〜60代） |
| 目的 | 採用LP制作サービスへの無料診断誘導 |
| 本数 | 15テーマ × 6本 = **90本** |
| 投稿開始 | v2完了後（目安：2026年9月17日〜） |

---

## v2との差分方針

v2（現在の90本）が「採用LPの必要性を認識させる」のに対し、
v3は**「導入後の運用・改善・深化」**の視点を中心に設計する。
→ 既存フォロワーへの深掘り＋新規への異なる角度での訴求
