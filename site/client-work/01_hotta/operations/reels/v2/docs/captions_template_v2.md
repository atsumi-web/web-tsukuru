# キャプションテンプレート集 v2.1（新90本対応）

## ハッシュタグ方針（2026年最新）

> Instagramは現在 **3〜5個** のハッシュタグを推奨。量より「コンテンツの内容をアルゴリズムに伝える」ことが目的。

**毎回必ず入れる共通3個：**
```
#建設業採用 #採用LP #人手不足
```
+ テーマ別に **1〜2個だけ追加** して最大5個。

---

## 使い方
1. 投稿するリールの **テーマ番号** を確認する
2. 対応するテンプレートをコピー
3. `【フック】` を実際のフック文（リールの冒頭1行）に書き換える
4. Meta Business Suite に貼り付けて予約投稿
