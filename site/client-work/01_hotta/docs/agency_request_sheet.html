【堀田建設様】採用ページ連携における技術要件・作業依頼書

Web担当者様

平素より堀田建設様のコーポレートサイト保守・運用にご尽力いただきありがとうございます。
この度、堀田建設様の採用戦略アップデートに伴い、採用導線を新設の「採用特化型LP」へ一元化することとなりました。

つきましては、コーポレートサイト側からのスムーズな連携のため、以下の技術的な作業と仕様のすり合わせをお願いしたく、資料を取りまとめました。

1. 目的と変更の概要

  • 目的： 最新の求職者ニーズ（スマホ中心・離脱防止・LINE連携）に合わせた導線の一本化と、GA4等を用いた厳密なアクセス解析網の再構築。
  • 概要： コーポレートサイト（www.hottacon.co.jp）内にある既存の採用関連ページへのアクセスを、すべて新設の採用LP（サブドメイン）へ統合させます。

2. 御社（Webご担当者様）へのお願い事項

コーポレートサイトのシステム（WordPress等）側で、以下の2点の作業の**実施可否の確認**をお願いいたします。
※ご注意：新LPは現在制作中のため、今すぐ設定の「実行」をお願いするものではございません。具体的な作業実施日（リンク・転送の切り替え日）は追ってお知らせいたしますので、今回はあくまで事前の仕様すり合わせとなります。

作業①：メニューリンクの張り替え

グローバルナビゲーション、フッターメニュー、トップページ内のバナー等に設定されている「採用情報」のリンク先を、すべて新しいLPのURLへ変更をお願いします。

  • 変更前： https://www.hottacon.co.jp/recruit/
  • 変更後： https://recruit.hottacon.co.jp/ （※LPのURL）
作業②：既存ページからの301リダイレクト設定

検索エンジン（Google等）経由で直接古いページにアクセスしたユーザーの離脱・404エラーを防ぐため、以下のディレクトリ配下へのアクセスを、新しいLPへ自動転送（301リダイレクト）するようサーバー側（.htaccess等）での設定をお願いします。

  • 対象ディレクトリ： https://www.hottacon.co.jp/recruit/ 以下すべて
  • 転送先： https://recruit.hottacon.co.jp/

※一時的なキャンペーンではなく恒久的な変更となりますため、302ではなく「301リダイレクト」にてお願いいたします。

3. 責任分界点（保守範囲）について

今後の保守・管理において、システムの干渉や責任の所在が不明確にならないよう、堀田建設様合意のもと、以下の通り管轄を明確に分割いたします。

管轄領域 担当 内容
コーポレートサイト（本体） 御社 既存システムの保守、サーバーインフラ維持、上記リダイレクト・リンク設定。
採用特化型LP（コンテンツ部） 渥美（私） LPの内容更新、表示崩れ対応、お問い合わせフォームの障害対応、GA4/UTMによるアクセス解析と改善。

※採用LP本体のコンテンツに関しましては、すべて私（渥美）の方で責任を持って一次対応・改修を行います。
（※当方のLPはWordPress等の動的CMSシステムに依存せず、純粋な静的コード（HTML/CSS/JS）にてフルスクラッチ構築しております。そのため、システムアップデートに伴う不具合やセキュリティ脆弱性等の「システム保守リスク」は発生いたしません。御社側に保守工数等のご負担をおかけすることは一切ございませんのでご安心ください）

4. 打ち合わせでの確認事項

本件につきまして、上記作業の「具体的な実施スケジュール」や、御社システム仕様上における設定の懸念点などがございましたら、お打ち合わせのお時間にてすり合わせをさせていただけますと幸いです。
