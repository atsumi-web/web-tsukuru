# [SOSHOU コンテンツボリューム拡充計画（Dr.Pedal完全同等レベル化）]

トップページと採用トップに加え、サイト全体の説得力を「Dr.Pedal水準（LP並の濃密な読み物）」に引き上げるための【下層ページ設計図】です。

## 決定事項 (Approved Plan)

すでに構築済みの `index.html` と `recruit.html` に加え、以下の下層ページ（ABOUT、INTERVIEW、CONTACT）の構成・ロジックを実装します。

### 【追加設計 1：代表メッセージ / ABOUT（`about.html`）】
単なる会社概要ではなく、代表の「建設業界を変えたい」という想いを長文で語る熱いページ。
- **代表挨拶**: AI生成した代表ポートレート（社長）を使用。
- **3つの約束（Values）**: 
  1. 「職人の命を最優先する（安全装備の会社負担）」
  2. 「家族を安心させる（安定した月給制・残業ゼロ）」
  3. 「未来の職人を育てる（3D図面や動画でのロジカルな指導）」
- **家族へのメッセージ**: 『彼氏さん、旦那さん、息子さんを必ず安全に家に帰します』という強いコピーを配置し、建設業特有の「危険だから辞めて」という家族からの反対（嫁ブロック等）を未然に防ぐ。

### 【追加設計 2：社員インタビュー（`recruit/interview-*.html`）】
若手・職長（異業種）・ベテランの3名に対し、生成いただいた各2枚の写真（ポートレート＋現場作業風景）をフル活用して「A4 1枚アンケート型」の鉄板レイアウトを構築する。
- **共通Q&A構成（4問）**:
  - Q1. 入社前、どんなことで悩んでいましたか？（前職の不満や漠然とした将来への不安）
  - Q2. 蒼翔に応募する際、不安だったことは？（未経験の壁、本当に残業がないのか等）
  - Q3. 入社の「決め手」は何でしたか？（面接での社長の熱意、機材が全部新品だった等）
  - Q4. 今、どんなやりがいを感じていますか？（給料が上がった、早く帰って子供と遊べる等）
- **ページ下部**: 他の社員インタビューへの導線（Dr.Pedal完全再現）。

### 【追加設計 3：エントリーフォーム（`contact.html`）】
採用求職者（toC）と元請け業者様（toB）で窓口を明確に分ける。
- **採用エントリー（toC）**: 「履歴書不要・まずは私服でヤキニクでも食べながら話そう」というスタンスで、心理的ハードルを極限まで下げる【LINE公式アカウントでエントリー】ボタンを巨大に配置。
- **協力業者・ご発注（toB）**: 企業向けのフォーマルなお問い合わせ・見積依頼フォーム。
