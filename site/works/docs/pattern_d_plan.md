# Pattern D: 透明性重視型「数字と社員の声で証明」実装計画

## コンセプト

**「理性（データ）と感情（声）の両面から、応募者の不安を完全に解消する」**

- Pattern A が「共感型」なら、Pattern D は「証明型」
- 疑い深い求職者、データ重視の求職者、家族の説得が必要な求職者に刺さる

---

## デザイン方針

### カラー

- **メインカラー:** Navy（信頼感）
- **アクセント:** Gold（認定・証明の強調）
- **データ表示:** グラフや数字をビジュアル化（緑・青系で安心感）

### レイアウト

- **データセクション:** カード型、大きな数字、アイコン付き
- **アンケートセクション:** 質問を見出しに、回答を箇条書き
- **余白:** たっぷり取って、読みやすく

## 必要な追加素材

   - 女性社員比率
   - 資格取得支援実績

2. **アンケート回答**

   - 上記 6 つの質問への社員（岡安、菅原、渥美）の回答

3. **ユースエール認定証の画像**

---

## 実装優先度

⭐⭐⭐ **最優先**

理由：

- Pattern A は「共感型」だが、データ重視の求職者には刺さりにくい
- Pattern D で「理性的な納得」を提供することで、幅広い層をカバーできる
- ユースエール認定という「国のお墨付き」は最強の信頼シグナル
