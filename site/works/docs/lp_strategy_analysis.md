# 堀田建設 採用 LP 戦略分析

## 現状の課題

現在の `plan_a/index.html` は、**3 つの異なるペルソナ**を同時に訴求しています:

| ペルソナ       | 特徴               | 主な不安・ニーズ                             |
| -------------- | ------------------ | -------------------------------------------- |
| **岡安タイプ** | 地元就職、定着型   | 「地元で働くことへの迷い」「キャリアの不安」 |
| **菅原タイプ** | 異業種転職、未経験 | 「業界知識ゼロからの不安」「第二のキャリア」 |
| **渥美タイプ** | 高卒、若手成長志向 | 「学歴コンプレックス」「資格取得・成長」     |

### メリット

- 幅広い層にリーチできる
- 「誰かには刺さる」可能性が高い

### デメリット

- メッセージが分散し、**誰にも深く刺さらない**リスク
- 各ターゲットの「最も知りたいこと」が埋もれる
- LP が長くなり、離脱率が上がる

---

## 戦略的選択肢

### 🅰️ 戦略 A:単一 LP(現状維持+最適化)

**コンセプト:** 「私たちが、続けている理由。」というテーマで、3 つのストーリーを統合

#### メリット

- 制作・管理コストが低い
- 「多様性」をアピールできる
- 広告の出し分けが不要

#### デメリット

- 各ターゲットへの訴求が弱まる
- コンバージョン率が中途半端になる可能性

#### 最適化案

1. **冒頭で選択肢を提示**

   ```
   あなたはどのタイプ?
   □ 地元で働くか迷っている
   □ 異業種から転職を考えている
   □ 高卒からキャリアを築きたい
   ```

   → クリックで該当インタビューにジャンプ

2. **共通の「核」を強化**
   - 「人間力」「働き続けられる環境」など、全員に共通する価値

---

### 🅱️ 戦略 B:ターゲット別 LP(3 分割)

**コンセプト:** ターゲットごとに専用 LP を作成

| LP 名           | ターゲット     | メインメッセージ                       | CTA                  |
| --------------- | -------------- | -------------------------------------- | -------------------- |
| **地元定着 LP** | 地元就職検討者 | 「地元で働く、誇りと安心」             | 地元出身者向け面談   |
| **転職者 LP**   | 異業種転職者   | 「未経験でも、ここなら育つ」           | キャリアチェンジ相談 |
| **若手成長 LP** | 高卒・新卒     | 「学歴じゃない。成長できる環境がある」 | 資格取得支援説明会   |

#### メリット

- **メッセージが鋭く刺さる**
- 各ターゲットの不安に直接答えられる
- コンバージョン率の向上が期待できる
- 広告の最適化がしやすい

#### デメリット

- 制作・管理コストが 3 倍
- 広告の出し分けが必要
- どの LP に誘導するか判断が難しい場合も

---

### 🅲️ 戦略 C:ハイブリッド型(推奨)

**コンセプト:** メイン LP(総合版) + 特化ページ(部分強化)

#### 構成

1. **メイン LP(現在の plan_a)**: 全ターゲット向け、会社の魅力を総合的に訴求
2. **特化セクション**: メイン LP 内に「あなた向けの情報」として分岐

#### 実装イメージ

```
[ヒーロー]
↓
[3つのタイプから選ぶ] ← NEW!
↓
┌─────────┬─────────┬─────────┐
│地元定着型 │転職者型  │若手成長型│
│(岡安)    │(菅原)    │(渥美)   │
└─────────┴─────────┴─────────┘
↓
[共通の環境・制度]
↓
[数字で見る働きやすさ]
↓
[CTA]
```

#### メリット

- **1 つの LP で完結**しつつ、ターゲット別訴求も可能
- 制作コストは 1.5 倍程度
- ユーザーが自分で「自分事化」できる

---

## ビジネスコーチの視点を加味した推奨戦略

### 🎯 推奨:**戦略 C(ハイブリッド型)**

理由:

1. **「人の顔と声」を活かせる** - 既存の 3 人のインタビューが強み
2. **無料モニター期間で検証できる** - 1 つの LP で複数パターンをテスト
3. **拡張性がある** - 反応が良いターゲットに後から特化 LP を作成可能

### 具体的な実装案

#### Phase 1:現在の LP を最適化(優先度:高)

- [ ] ヒーロー直下に「あなたはどのタイプ?」診断を追加
- [ ] 各インタビューセクションの冒頭に「こんな人におすすめ」を明記
- [ ] 「人間力」というコアメッセージを追加
- [ ] 1 日の業務フローを追加(ビジネスコーチ指摘)

#### Phase 2:データと信頼性の強化(優先度:中)

- [ ] 「数字で見る堀田建設」セクションを充実
- [ ] 働き方改革・若者会議などの取り組みを追加
- [ ] 地域貢献(避難場所としての本社)をストーリー化

#### Phase 3:効果測定と改善(優先度:中)

- [ ] どのタイプが最も反応が良いかを分析
- [ ] 必要に応じて特化 LP を別途作成
