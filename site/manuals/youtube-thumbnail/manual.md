# YouTubeサムネイル制作マニュアル

## Adobe Fireflyを使った3ファイル方式

> **このマニュアルで誰でも同品質のサムネイルが作れます**

---

## 📁 STEP 0: 素材を3つ揃える

| ファイル名             | 内容                       | 入手方法                      |
| ---------------------- | -------------------------- | ----------------------------- |
| `デザイン参考.jpg`     | 真似るデザインのサムネイル | 参考サイトからダウンロード    |
| `プロフィール.jpg`     | 差し替える人物の写真       | クライアントから受け取る      |
| `現在のサムネイル.jpg` | テキスト内容の参考         | クライアントのYouTubeから取得 |

> 📌 **参考サイト**
>
> - サムネギャラリー: https://thumbnail-gallery.net/category/genre/business-education
> - ダウンロードツール: https://wiz.ooo/apps/yt-thumbnail-dl

---

## 🖥️ STEP 1: Fireflyを開く

1. [Adobe Firefly](https://firefly.adobe.com/) にアクセス
2. Adobeアカウントでログイン
3. 「**テキストから画像**」を選択

---

## 🖼️ STEP 2: 参照画像をセットする

| 参照の種類       | セットするファイル |
| ---------------- | ------------------ |
| **被写体参照**   | `プロフィール.jpg` |
| **スタイル参照** | `デザイン参考.jpg` |

---

## 📋 STEP 3: 指示書をコピペして貼り付ける

下記テンプレートをコピーして、`[ ]`の中を書き換えて使ってください。

```
デザイン参考.jpgをベースにして、
人物画像をプロフィール.jpgに差し替えてください。

"[元のテキスト①]"
を
"[新しいテキスト①]"
に変更してください。

"[元のテキスト②]"
を
"[新しいテキスト②]"
に変更してください。
```

---

### ✅ 記入例（実際の案件より）

```
デザイン参考.jpgをベースにして、
人物画像をプロフィール.jpgに差し替えてください。

"MOTIVATION"
を
"WEB DESIGN"
に変更してください。
```

---

## ✨ STEP 4: 生成して確認する

1. 「**生成**」ボタンをクリック
2. 4枚の候補から一番良いものを選ぶ
3. 気に入らない場合は「**もう一度生成**」
4. 良いものが出たら**ダウンロード**（PNG推奨）

---

## 🎨 STEP 5: 背景透過が必要な場合

1. Adobe Expressを開く → https://express.adobe.com/
2. ダウンロードした画像をアップロード
3. 「**背景を削除**」をクリック（1秒で完了）
4. 透明PNG でダウンロード

---

## ✅ 納品前チェックリスト

- [ ] テキストの誤字・表記ゆれがない
- [ ] 人物の顔が自然に合成されている
- [ ] デザイン参考のカラーテーマが維持されている
- [ ] 1280×720px（YouTube標準サイズ）になっている
- [ ] ファイル形式はJPGまたはPNG

---

## 🚨 よくある失敗と対処法

| 問題                   | 対処法                                   |
| ---------------------- | ---------------------------------------- |
| テキストが変わらない   | Before/Afterを「"　"」で囲んで明示する   |
| 人物が不自然           | プロフィール.jpgを背景透過してからセット |
| 全体のデザインが崩れる | 指示を1つずつ分けて送る                  |
| 画質が低い             | Upscale.mediaで解像度アップ後に納品      |
