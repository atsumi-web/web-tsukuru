## 🔍 「ここが惜しい」分析（アプローチ文の核心）

| 課題                         | 詳細                                                                             |
| ---------------------------- | -------------------------------------------------------------------------------- |
| ① SNSと採用ページの完全分断  | Instagram/TikTokで「熱量」を発信しているのに、採用ページへのリンク・バナーがゼロ |
| ② 採用ページが情報の羅列     | 募集要項のみ。社員の顔・インタビュー・働くイメージが一切ない                     |
| ③ 条件の良さが伝わっていない | 奨学金返還・年休125日など競合優位な条件があるのに、魅力的なコンテンツ設計がない  |
| ④ 若手へのビジュアル訴求ゼロ | SNSで現場感を発信しているのに、採用導線上でそれを受ける場所がない                |
