# お問い合わせ対応テンプレート集 for LINE/メール

「Web をつくるしごと（採用特化 LP 制作）」へのお問い合わせに対する返信テンプレートです。
**全業対応（建設業に限らず、飲食・サロン・IT など）**を想定した汎用的な内容になっています。

コピー＆ペーストして、[ ] の部分を書き換えてご使用ください。

---

## 1. 初回お問い合わせ対応

### パターン A：興味あり / 詳細を聞きたい

**【使用シーン】** 「LP 制作に興味があります」「話を聞いてみたい」などの連絡が来た時。
**【ゴール】** 無料相談（Zoom ヒアリング）のアポを取ること。

```text
[お名前] 様

はじめまして！
「Webをつくるしごと」のあつみ えりと申します。
この度は、お問い合わせいただきありがとうございます！😊

採用LPの制作にご興味を持っていただけて嬉しいです。
当社では、まず詳しくお話をお伺いして、御社にぴったりの構成をご提案させていただいております。

もしよろしければ、一度30分〜1時間ほどオンライン（Zoom）でお話しませんか？
「まだ依頼するか決めていない」という段階でも全く問題ありません！✨

以下の日程でご都合はいかがでしょうか？

【候補日】
・〇月〇日（　）10:00〜18:00
・〇月〇日（　）13:00〜16:00
・〇月〇日（　）10:00〜15:00

※上記以外でも調整可能ですので、ご希望の日時があれば教えてください。
お返事お待ちしております！📩
```

### パターン B：料金についての問い合わせ

**【使用シーン】** 「費用はいくらですか？」「料金表を見たい」といきなり聞かれた時。
**【ゴール】** 概算を伝えつつ、やはりヒアリングに誘導して価値を伝えること。

```text
[お名前] 様

お問い合わせありがとうございます！
「Webをつくるしごと」のあつみです。

費用の件ですね、ご検討いただきありがとうございます。

当社の採用LP制作は、取材・撮影・ライティング・デザイン・構築まですべて含んだ「丸投げプラン」で、
基本料金 **[金額]円（税別）** にて承っております。

ただ、必要なページ数やオプション（動画制作など）によって、最適なプランが変わることもございます。
一度、現状の課題やご要望をお伺いした上で、正式なお見積りを出させていただきたく存じます。

無理な営業は一切いたしませんので、まずは一度カジュアルにお話しませんか？😊

【無料相談の候補日】
・〇月〇日（　）10:00～
・〇月〇日（　）14:00～
・〇月〇日（　）16:00～

ご都合の良い日時を教えていただければ幸いです！
```

## 2. 日程調整・リマインド

### 日程確定の連絡

**【使用シーン】** 相手から候補日の返信が来た時。
**【ゴール】** Zoom URL を送って確定させる。

```text
[お名前] 様

ありがとうございます！
では、以下の日時で予約させていただきますね。

📅 日時： 〇月〇日（　） 14:00〜
💻 方法： オンライン（Zoom）

当日はこちらのURLからお入りください。
[ZoomのURLを貼り付け]

画面共有をしながら、過去の制作事例などもお見せできればと思います。
当日お話しできるのを楽しみにしております！✨
```

### 前日リマインド（超重要！）

**【使用シーン】** ヒアリングの前日。
**【ゴール】** ドタキャン防止。

```text
[お名前] 様
お世話になっております、あつみです。

明日のオンライン相談の件、ご連絡いたしました！

📅 日時： 明日 〇月〇日（　） 14:00〜
💻 URL： [ZoomのURL]

もし急なご予定が入ってしまった場合は、遠慮なくおっしゃってくださいね。
（別の日程で再調整いたしますので！👍）

それでは明日、よろしくお願いいたします！
```

## 3. ヒアリング後のフォロー

### お礼メール（当日中）

**【使用シーン】** Zoom 終了後、すぐ送る。
**【ゴール】** 信頼感アップと、次のステップ（契約・見積もり）への誘導。

```text
[お名前] 様

本日はお忙しい中、貴重なお時間をいただきありがとうございました！
あつみです。

[お名前]様の想いや、[会社名]様の今後のビジョンについてお伺いでき、私も大変ワクワクしました✨
特に、[ヒアリング等の話題を1つ入れる] のお話はとても勉強になりました。

本日お話しした内容を元に、最適な構成案とお見積りを作成いたします。
〇日以内に改めてご提案させていただきますので、少々お待ちくださいませ。

まずは取り急ぎ、本日のお礼まで。
引き続きよろしくお願いいたします！
```

### 検討状況の確認（3 日後〜1 週間後）

**【使用シーン】** 提案後、返信がない時。
**【ゴール】** 忘れられていないか確認する（しつこくしない）。

```text
[お名前] 様
お世話になっております、あつみです。

先日はありがとうございました！
その後、ご検討状況はいかがでしょうか？

もし「ここが懸念点」「もう少し安くならないか」など、
ご相談があれば柔軟に対応させていただきますので、いつでもお声がけくださいね😊

ご多忙と存じますので、ご返信はお手隙の際で構いません。
よろしくお願いいたします。
```

## 💡 その他の対応

### 業種について聞かれた場合（建設業以外）

**【質問】** 「飲食店（or 美容室、IT など）ですが対応できますか？」
**【回答】** 本質（採用の悩み）は同じなので「大歓迎」と伝えます。

```text
はい、もちろんです！✨

当社は「建設業」の実績も多いですが、
「求職者に会社の魅力を正しく伝える」という採用の本質はどの業界も共通ですので、
飲食・美容・IT・医療など、どんな業種でも対応可能です！

むしろ、異業種の成功事例を取り入れることで、他社と差別化できるいいLPができるはずです。
ぜひ詳しくお話を聞かせてください！😊
```
