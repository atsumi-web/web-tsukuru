# LINE 公式アカウント 構築計画

## 1. 目的・役割

- [x] **問い合わせ窓口**: Web 制作に関する相談・見積もり依頼
- [x] **進行管理・連絡**: 契約後のクライアントとのやりとり
- [ ] **情報発信**: 制作事例やコラムの配信（メルマガ的用途） → 今後検討

## 2. アカウント基本情報

- **アカウント名**: Web をつくるしごと (仮)
- **ID (ベーシック ID/プレミアム ID)**: 自動発行されます（後で任意 ID 購入も可能）
- **対象ターゲット**: "Web をつくるしごと" の見込み顧客
- **アイコン**: ロゴ（Web をつくるしごと）を使用

## 3. よくある質問 (FAQ)

### Q. ビジネス用に新しい携帯番号は必要ですか？

**A. いいえ、不要です。**
個人の LINE アカウント（またはメールアドレス）を使って「LINE 公式アカウント」を開設・管理できます。
個人のトーク画面とは完全にアプリ（または管理画面）が分かれるため、友だち（クライアント）側には、えりさんの個人の LINE ID やプライベートな情報は一切見えません。
※管理用に「LINE Official Account Manager」という別アプリ（Web 版もあり）を使います。

## 4. 制作が必要なクリエイティブ

LINE 公式アカウントの見栄えを整えるための素材です。

- [x] **プロフィール画像**: ロゴアイコン (丸型にトリミングされます)
- [ ] **背景画像 (カバー画像)**: アカウントのトップページ上部
- [ ] **リッチメニュー**: トーク画面下部の固定メニュー（Web サイトへのリンク、料金表、実績など）
  - ※まずは「リッチメニューなし」で開始も可能ですが、あるとプロ感がでます。

## 5. 推奨設定フロー

### STEP 1: アカウント開設

- LINE Official Account Manager でアカウントを作成。

### STEP 2: 初回メッセージ（あいさつメッセージ）の作成

- 友だち追加直後に送られるメッセージ。
- 「ご登録ありがとうございます。Web 制作のご相談はこちらからどうぞ」等の案内。

### STEP 3: プロフィールページの設定

- 基本情報、Web サイト URL、営業時間などの設定。

### STEP 4: Web サイトへの設置

- `web-tsukuru` の LP 等のコンタクト導線に LINE 追加ボタンを設置。
