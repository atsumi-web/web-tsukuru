# 日報作成マニュアル

**目的**: 毎日5分で進捗を記録し、今日やることを明確にする。

---

## 📋 毎朝やること（1つだけ）

### Step 1: Antigravity にプロンプトを投げる

以下をコピーして、Antigravity チャットに貼り付けてください。

```
@Antigravity
おはよう。昨日のチャットログを読んで、

1. 昨日進んだことの整理（日報作成）
   - 90日ロードマップに基づいた項目だけピックアップ
   - 細かい作業は省いて、大きな成果や進捗だけ記録
   - 「こんなことやりましたよね」という形で候補を挙げてから日報作成

2. 今日の `manuals/today.html` サブタイトル提案（1行コピペ用）

3. 今日やるべきことの洗い出し（TODAY.md更新）

この3つをやってほしい。
頭が冴えている今、戦略的に動きたいので、90日ロードマップに基づいた鋭い提案を頼む。
```

### Step 2: 今日のミッションを確認

Antigravity が TODAY.md を更新したら、以下を開く：

**https://web-tsukuru.jp/manuals/today**
