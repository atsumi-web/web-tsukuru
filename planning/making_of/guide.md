# メイキング記録ガイドライン

## 目的

「25 万円の価値」を証明するために、完成品だけでなく「制作の裏側（プロセス）」を可視化・資産化する。
完成した Web サイトだけでは伝わらない「戦略」「思考」「試行錯誤」の量を見せることで、顧客に「ここまでやってくれるなら安い」と感じてもらうことがゴールです。

---

## 📁 記録するものリスト

### 1. 脳内・思考の跡（★ 最重要）

**「形になる前」の泥臭い検討プロセスこそが最大の価値です。**

- [ ] **手書きメモ・ラフ**: ノート、裏紙への書き殴り、iPad のスケッチなど
- [ ] **構成案の推敲**: 付箋を貼り付けたアナログな検討資料、赤入れした原稿
- [ ] **マインドマップ**: 思考を整理しているホワイトボードや紙

### 2. デジタル作業の裏側

**「AI 使いこなし」や「緻密な設計」をアピールします。**

- [ ] **AI との対話ログ**: Gemini と壁打ちして戦略を練っている画面（「こんなに深く考えている」証拠）
- [ ] **フォルダ構成**: 整理整頓されたプロジェクトフォルダのスクショ（几帳面さ、安心感）
- [ ] **デザインのボツ案**: 採用されなかったが検討した別パターン（「複数案から選び抜いた」証拠）
- [ ] **コード画面**: VS Code 等の作業画面（「プロっぽい」見た目のインパクト）

### 3. "人"が見える瞬間

**「誰が作っているか」の体温を伝えます。**

- [ ] **作業風景**: パソコンに向かっている姿、カフェでの作業、デスクの手元（自撮りやタイマー使用）
- [ ] **打ち合わせ風景**: Zoom 画面のスクショ、対面ヒアリングの様子
- [ ] **愛用品**: お気に入りのペン、マウス、休憩中のコーヒー

## 📸 撮影のタイミング（意識付け）

1. **作業を始める前**に一枚（「これからやるぞ」）
2. **煮詰まった時**に一枚（思考中のメモや画面）
3. **完成した時**に一枚（成果物とドヤ顔）

> **💡 ポイント**
> 写真は「綺麗」である必要はありません。むしろブレていたり、影が入っていたりする「生々しさ」がリアリティを生みます。迷わずどんどん撮ってここに放り込んでください！
