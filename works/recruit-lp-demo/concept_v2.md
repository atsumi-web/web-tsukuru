# 地域密着型・高付加価値採用 LP リブランディング計画書 (Ver.2.0)

## 1. ターゲット顧客（クライアント）の再定義

### クライアント像：地域を支える「地場ゼネコン」

- **想定企業:** 髙田建設株式会社（創業 40 年、社員数 30 名規模）
- **事業内容:** 地域内の公共施設（学校・役所）、マンション、商業ビルの建築（RC 造・S 造メイン）。
- **商圏:** 車で 1 時間圏内のエリア限定（地域密着）。転勤なし。
- **現状の課題:**
  - 「大手の下請け」「古臭い」と思われがちで、若手採用に苦戦している。
  - 実は公共工事のランクが高く経営は超安定、給与も悪くないが、それが知られていない。
  - Web サイトが 10 年以上前のもので、スマホで見づらく、応募に至らない。
- **社長の想い:**
  - 「うちは『きつい・汚い』現場じゃない。ICT も入れてるし、ちゃんと休める。今の若い子たちにも、この仕事の『誇り』と『働きやすさ』を知ってほしい」

## 2. デザイン・コンセプト：Local Pride, Global Quality

**「泥臭い地場工務店」ではなく、「地域で一番かっこいいプロ集団」を目指す。**
ターゲットとなる求職者（地元の若者・マイルドヤンキー層・堅実派）は、必ずしも「都会の有名企業」で働きたいわけではなく、「地元で、安定して、かっこよく稼ぎたい」と考えている。

### キーワード

- **Reliable (信頼感):** 軽薄なポップさではなく、重厚感と安心感のあるデザイン。
- **Modern (先進性):** 「古い業界」のイメージを覆す、洗練されたタイポグラフィとレイアウト。
- **Local Hero (地域の主役):** 「俺たちがこの街を作っている」という自負を感じさせるビジュアル。

## 3. 構成・コンテンツの修正案

現在の「TETO インスパイア」デザインの高いクオリティは維持しつつ、コピーと訴求ポイントを「地域密着」に最適化します。

| セクション     | 修正方針                           | 具体的な変更イメージ                                                                                                                                                        |
| :------------- | :--------------------------------- | :-------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Hero (MV)**  | **「地域性」と「プロ感」の融合**   | キャッチコピーを**「この街の景色は、俺たちがつくる。」「転勤なし。地元で、一流の仕事をする。」**といった、"地元 × 誇り"を感じさせるものに変更。                             |
| **Merit**      | **「働きやすさ」の解像度を上げる** | 「ICT 活用」などの先進性は残しつつ、「**現場は車で 30 分圏内（移動が楽）**」「**転勤なし（ずっと地元で暮らせる）**」「**直行直帰 OK**」など、生活に密着したメリットを強調。 |
| **Atmosphere** | **「大規模 × チームワーク」**      | 先ほど修正した「RC/大規模修繕の現場」の画像を使用。戸建てばかりではなく、「地域のランドマーク」を手掛けている規模感を出すことで、仕事へのやりがいを演出。                   |
| **Message**    | **「社員＝家族」の温度感**         | 「職人の使い捨てをなくす」という強いメッセージに加え、「社員とその家族まで大切にする」「地域に育ててもらった恩返し」という、地場企業ならではの温かみをプラス。              |
| **Interview**  | **等身大の先輩たち**               | 「未経験から家を建てた」「地元の祭りに参加できるようになった」など、仕事とプライベートの両立や、地域での生活の充実感を語ってもらう。                                        |

## 4. この設計図の狙い

この設計図に基づいて LP を修正することで、以下の効果を狙います。

1.  **競合（他の地場建設会社）との圧倒的な差別化:** 周りと同じような古いサイトではなく、一見して「ここは他と違う」と思わせるブランド力を付与。
2.  **ターゲット人材の共感:** 「地元で働きたいけど、ダサいのは嫌だ」という若者のインサイトを突き、応募のハードルを下げる。
3.  **クライアント（社長）の満足度:** 潜在的に持っていた「自社への誇り」を可視化することで、作成サービス自体の価値（単価）を高める。
