## Pattern A: 現状改善型「続けている理由」

### コンセプト

現在の plan_a をベースに、ユーザーが「なぜ自分が応募すべきか」を自己発見させる要素を追加

### ターゲット

- 安定と成長のバランスを求める求職者
- 建設業界への不安を持つ未経験者

### 独自要素

- **「あなたはどのタイプ？」診断ツール** → 自分ごと化
- **3 人の深掘りインタビュー** → 共感・ストーリー型
- **ユースエール認定バッジ** → 信頼性の担保

### 感情の流れ

不安 → 共感 → 自己発見 → 納得 → 応募

---

## Pattern B: キャプテン育成型「最高のリーダーへ」

### コンセプト

社長のビジョンと育成制度を前面に、成長ストーリーを描く

### ターゲット

- 成長意欲が高い人
- リーダーシップを発揮したい人
- 野心的な若手

### 独自要素

- **「鬼。悪魔。でも時々、神」** → 強烈なキャッチコピー
- **若者会議の紹介** → 意見を言える環境の証明
- **社長インタビュー** → トップの本気度を見せる

### 感情の流れ

現状への不満 → 成長への渇望 → ビジョンへの共感 → 挑戦意欲 → 応募

---

## Pattern C: U ターン特化型「地元で、プロとして」

### コンセプト

地元愛と専門性を両立させる働き方を提案

### ターゲット

- U ターン・J ターン希望者
- 地元出身者
- 経験者（他社からの転職）

### 独自要素

- **林さんの三重大学エピソード** → ロールモデル提示
- **地元プロジェクト実績** → 地域への貢献を可視化
- **長期メンテナンス** → 地元に根ざす仕事の意義

### 感情の流れ

地元への愛着 → キャリアへの不安 → プロとしての可能性発見 → 納得 → 応募

---

## Pattern D: 透明性重視型「数字と社員の声で証明」

### コンセプト

理性（データ）と感情（声）の両面から、応募者の不安を完全に解消

### ターゲット

- データ重視の求職者
- 疑い深い人
- 家族の説得が必要な人

### 独自要素

- **ユースエール認定を大きく強調** → 国のお墨付き
- **数値データの可視化** → 離職率 0%、残業 16.6h、有給取得率など
- **全員アンケート結果** → 統計的信頼性

### 感情の流れ

疑念 → データによる納得 → リアルな声による共感 → 信頼 → 応募

---

## Pattern E: 地域貢献型「100 年後の誇り」

### コンセプト

社会貢献・地域への影響を強調し、仕事の意義を訴求

### ターゲット

- 社会貢献意識が高い人
- 長期的視点で仕事を選ぶ人
- 「意味のある仕事」を求める人

### 独自要素

- **亀山中学校エピソード** → 地域とのつながり
- **便利屋精神** → 困っている人を助ける文化
- **避難場所としての本社など** → 地域インフラとしての役割

### 感情の流れ

漠然とした不安 → 仕事の意義の発見 → 地域への貢献イメージ → 誇り → 応募

## 5 パターンの使い分け戦略

### 1. 広告・SNS での出し分け

- **Pattern A**: 汎用性が高い → 一般的な求人媒体
- **Pattern B**: 「成長」「キャリア」キーワードで興味を持った人向け
- **Pattern C**: 「地元」「U ターン」キーワード検索者向け
- **Pattern D**: 「ホワイト企業」「働きやすさ」検索者向け
- **Pattern E**: 「社会貢献」「やりがい」検索者向け

### 2. AB テストでの検証

- 同じ予算で 5 パターンを並行配信
- コンバージョン率・滞在時間・離脱率を比較
- 最も効果的なパターンに予算を集中

### 3. プレゼン時の提案

クライアントに「どのターゲットを最優先するか」のヒアリングを行い、最適なパターンを推奨
