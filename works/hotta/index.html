BUILDING 建築部 現場監督

K.O（26歳 / 入社4年目 / 大卒）

「1年目から、現場の最前線へ。
理論だけでなく、実践で学ぶ場がある。」

入社の決め手は？

大学で建築を学び、地元で大きな仕事がしたいと思っていました。面接で社長の「若手にチャンスを与える」という言葉に熱意を感じ、ここなら自分の知識を活かして早く成長できると確信しました。

仕事のやりがいは？

自分が図面から管理した建物が、実際に形になり、地図に残ることです。苦労もありますが、完成引き渡し時のお客様の笑顔を見ると、すべての疲れが吹き飛びます。

BUILDING 建築部 主任

S.Y（25歳 / 主任 / 転職）

「商業高校から、別業界を経てここへ。
実力主義で、20代で主任になれました。」

現場の雰囲気は？

前職とは違い、ここでは「見て覚えろ」ではなく、先輩が手厚くサポートしてくれます。僕は商業高校出身で建築は未経験でしたが、やる気を評価してもらい、25歳で現場を任されるまでになりました。

これからの目標は？

主任として、後輩の育成にも力を入れたいです。自分が先輩にしてもらったように、次は自分が若手を引っ張り、会社一番のチームを作ることが目標です。

BUILDING 建築部 現場監督

M.T（21歳 / 入社3年目 / 高卒）

「高卒で飛び込んだ新しい世界。
毎日が発見と成長の連続です。」

建設業を選んだ理由は？

高校卒業後、手に職をつけたくて入社しました。最初は不安だらけでしたが、上司や先輩が娘のように大切にしてくれて（笑）。今では現場のアイドル...ではなく、一人の戦力として認めてもらっています。

働きやすさはどうですか？

女性が少ない現場ですが、更衣室やトイレも専用のものを用意してくれますし、重いものは男性陣がサッと持ってくれる優しさがあります。安心して働ける環境です。
