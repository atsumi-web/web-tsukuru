## プロジェクト概要

### 目的

堀田建設様の採用活動を強化するため、「使い捨てではない、一生使える採用資産」としてのランディングページを構築。

### 戦略

1. **3つのペルソナによる多様なストーリー**
2. **透明性の高いFAQ**
3. **地元密着の価値を前面に**
