# Pattern B: 感情設計フロー

## 読者の感情の旅

### Phase 1: 共感(不安の言語化)

**読者の心理**: 「自分に建設業なんてできるのか?」「地元で働くのは逃げなのか?」

**セクション**: オープニング

```
見出し: 「正直に言おう。迷ってるだろ?」

本文:
地元で働くか、都会に出るか。
建設業に挑戦するか、安定した道を選ぶか。
未経験でやっていけるのか、学歴がなくて大丈夫なのか。

その迷い、全部、わかる。
俺たちも、最初から覚悟があったわけじゃない。
```

**感情**: 「この会社、自分のこと分かってくれてる」

---

### Phase 2: 問題提起

**読者の心理**: 「でも、どうすればいいんだ?」

**セクション**: 問いかけ

```
見出し: 「でも、聞かせてくれ。」

本文:
お前は、どんな大人になりたい?
誰かの指示を待つだけの歯車か?
それとも、チームを率いて、何かを成し遂げるキャプテンか?

建設業は、「指示待ち」では務まらない。
現場をまとめ、職人をリスペクトし、お客様の夢を形にする。
それができるのは、「人間力」を持ったキャプテンだけだ。
```

**感情**: 「そうだ、自分は成長したいんだ」

---

### Phase 3: 希望の提示(社長のビジョン)

**読者の心理**: 「でも、自分にそんな力あるのか?」

**セクション**: 約束

```
見出し: 「鬼。悪魔。でも時々、神。」

本文:
うちの専務は、鬼だ。悪魔だ。
厳しい。容赦ない。プレッシャーもすごい。

でも、時々、神になる。

「チャレンジして失敗するのは、素敵だ。応援したくなる。」
そう言って、お前の挑戦を、全力で支える。

スキルも学歴も、今は、いらない。
ただ、「最高のキャプテンになりたい」という、
熱い向上心だけを持って、うちの門を叩いてくれ。

そうすれば、俺たち全員で、お前を、
代替不可能な「人財」に、絶対に、育て上げてみせる。
```

**感情**: 「ここなら、自分も変われるかもしれない」

---

### Phase 4: 証拠(ストーリー)

**読者の心理**: 「本当に?嘘じゃない?」

**セクション**: 3 つのストーリー

#### Story 1: 林さん(人間力の証明)

```
見出し: クレームから、ケーキへ。

導入: 警察沙汰になるほどのクレーム現場。
展開: 諦めずにコミュニケーションを取り続けた。
結果: 「ケーキ作ったから取りにおいで」と言われるまでに。

メッセージ: 技術だけじゃない。人間力で、現場を変えられる。
```

#### Story 2: 亀山中学校(成長の証明)

```
見出し: 緊張が、自信に変わる瞬間。

導入: 構造一級建築士との厳しい検査。
展開: 緊張しながらも万全の準備で臨んだ。
結果: 検査に合格し、プロと対等に渡り合えた。

メッセージ: プレッシャーの先に、成長がある。
```

#### Story 3: 専務の言葉(文化の証明)

```
見出し: 失敗は、素敵だ。

メッセージ: 挑戦を応援する文化がある。
```

**感情**: 「本当に成長できる環境なんだ」

---

### Phase 5: 客観的証拠(データ)

**読者の心理**: 「でも、ブラック企業じゃないよね?」

**セクション**: ユースエール認定

```
見出し: 離職率0%。その理由。

データ:
- 直近3年の新卒離職率 0%
- 月平均残業時間 16.6時間
- 有給取得平均日数 15日以上
- ユースエール認定企業(国が認めた優良企業)
```

**感情**: 「安心して働ける環境だ」

---

### Phase 6: 最後の後押し

**読者の心理**: 「でも、まだ不安...」

**セクション**: 最後のメッセージ

```
見出し: 一緒に、この会社の、次の10年を創ろうじゃないか。

本文:
お前が今、感じている不安。
それは、成長の予兆だ。

完璧な人間なんて、いらない。
未完成だからこそ、伸びしろがある。

覚悟は、決まったか?
```

**感情**: 「よし、やってみよう」

---

### Phase 7: 行動喚起(CTA)

**読者の心理**: 「どうすればいい?」

**セクション**: CTA

```
見出し: まずは、話を聞いてみないか?

選択肢:
1. 会社見学・現場見学
2. 会社説明会
```

**感情**: 「まずは一歩踏み出してみよう」

---

## 実装の順序

1. オープニング(共感)
2. 問いかけ(問題提起)
3. 社長のビジョン(希望)
4. 3 つのストーリー(証拠)
5. データ(客観的証拠)
6. 最後のメッセージ(後押し)
7. CTA(行動喚起)
