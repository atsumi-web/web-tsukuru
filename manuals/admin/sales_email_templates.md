# 営業メールテンプレート（サイト診断レポート添付型）

## 基本方針

「無料モニター募集」ではなく、**「勝手に診断しました。価値を提供します」**という姿勢で、他の営業と差別化する。

## 📧 メールテンプレート（基本版）

### 件名

```
【無料診断】御社の採用サイトを分析させていただきました｜若手採用の課題が見えました
```

### 本文

```
株式会社〇〇〇〇
採用ご担当者様

**勝手ながら採用サイトの診断をさせていただきました。**

ぜひ一度ご覧いただけますと幸いです。

【診断結果のポイント】
① スマホ対応：20代の求職者は100%スマホで見ています
② 「人の顔」が見えるか：応募率を左右する最重要ポイント
③ 感情訴求：「なぜこの会社で働くべきか」が伝わっているか

特に、御社の強みである「〇〇〇（例：地域密着50年の実績）」を、
もっと若手に響く形で伝えることで、
応募数は2〜3倍に増えると確信しております。

現在、建設業界の採用課題解決のため、
**無料モニター企業様を3社限定で募集**しております。

もしご興味がございましたら、
お気軽にご返信いただけますと幸いです。

何卒よろしくお願いいたします。

〇〇〇〇
```

### 3日後に返信がない場合

- Instagram/Facebook DMで追撃
- 「先日メールをお送りしましたが、ご覧いただけましたでしょうか？」

### 1週間後も返信がない場合

- 一旦保留
- 1ヶ月後に再アプローチ

---

## 🎯 成功のポイント

1. **診断レポートの質が全て**
   - 具体的な改善案を提示
   - 「この人はプロだ」と思わせる

2. **相手の強みを褒める**
   - 「毎月更新されている」「100年の歴史」など
   - 上から目線にならない

3. **無料モニターの理由を明記**
   - 「実績作りのため」と正直に

4. **返信のハードルを下げる**
   - 「ご興味があれば」「お気軽に」
   - 売り込み感を出さない
