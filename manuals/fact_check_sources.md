## 5. Cloudflare Pages × GitHub の普及状況（日本国内）

### 市場分析サマリー

**結論**: まだ少数派だが、急速に注目を集めているブルーオーシャンな構成

**分析日**: 2026-02-11

### 日本国内のウェブ制作市場の二極化

| 層                   | 構成                                                            | シェア  | 特徴                                       |
| -------------------- | --------------------------------------------------------------- | ------- | ------------------------------------------ |
| 主流（マジョリティ） | レンタルサーバー（エックスサーバー、ConoHa WING等）＋ WordPress | 80%以上 | 圧倒的なシェア                             |
| 先進層（モダン開発） | Cloudflare Pages / Vercel ＋ GitHub                             | 少数派  | スタートアップや技術力の高い制作会社が採用 |

### 普及していない理由（＝ビジネスチャンス）

| 理由             | 内容                                                                                                 | 示唆                               |
| ---------------- | ---------------------------------------------------------------------------------------------------- | ---------------------------------- |
| 技術的ハードル   | 多くの制作担当者が「FTPでファイルを上げる」手法に慣れており、GitHub（Git）を使いこなせる層が限られる | 技術力のアピールポイントになる     |
| WordPress信仰    | 日本では「自分で更新したい＝WordPress」という図式が根強い                                            | LP（静的サイト）というニッチに最適 |
| 保守費用のモデル | 制作会社にとって「サーバー管理代」は貴重な継続収入源。無料サービスは利益モデルと合わない             | 顧客にとっては大きなメリット       |

### LP制作サービスとしての戦略的優位性

1. **圧倒的な差別化**:
   - 競合: 「月額数千円のレンタルサーバー」
   - 当社: 「世界標準のインフラをサーバー代無料で提供」

2. **「最新技術」というブランディング**:
   - GitHubによるバージョン管理 → 制作の透明性と信頼性
   - Cloudflare → 「古いレンタルサーバーではなく、最新のグローバルインフラ」

3. **先行者利益**:
   - 「みんなが使っていない」＝性能が悪いのではなく、「使いこなせる人が少ない」
   - この構成でLP制作サービスを行うことは、先行者利益を得られる非常に賢い戦略

### 営業トークへの活用

**クライアントへの説明例**:

> 「一般的な制作会社は、月額数千円のレンタルサーバーを使います。しかし当社は、世界トップ企業（Discord、JAL等）が採用するCloudflareという最新インフラを使っています。サーバー代は無料、表示速度は圧倒的、セキュリティも最高峰。これが当社の技術力です。」

#### LP制作サービスとしての分析と評価

1. **商用利用の完全解禁**:
   - Vercelは無料枠（Hobby）での商用利用を明確に禁止
   - Cloudflareは無料プランでもビジネス利用が可能 → 制作側も安心して顧客に提供できる

2. **「転送量無制限」という強力な武器**:
   - 他社: 広告運用でアクセス急増時、高額な追加料金やサイト停止のリスク
   - Cloudflare: 無制限 → 「どれだけアクセスが来ても追加料金なし」という強い安心感

3. **世界トップクラスのインフラ**:
   - セキュリティと表示速度において競合他社に劣ることはない

**結論**: LP制作サービスを行うのであれば、Cloudflare Pages一択。「サーバー代永久無料・転送量無制限・世界最高峰のセキュリティ」という3点は、顧客への強力なセールスポイント。

### なぜCloudflareは高性能なサービスを無料で提供できるのか？

**よくある質問への回答**: 「無料で大丈夫？怪しくない？」

世界的な上場企業であるCloudflareがこの戦略をとっているのには、明確なビジネス上の理由が3つあります。

#### 1. 巨大なインフラの「余剰リソース」の活用

- Cloudflareは、**全世界のインターネットトラフィックの約20%を処理**する巨大なネットワークを既に構築
- 個人のLPや小規模なWebサイトをホスティングすることは、「広大な海にコップ一杯の水を注ぐ」ようなもの
- **世界330拠点以上のサーバーの余った処理能力**を使っているため、追加の運用コストが極めて低い

#### 2. 「開発者の囲い込み」戦略（フリーミアムモデル）

**エコシステムへの導入**:

- まず無料でPagesを使ってもらい、サイトが成長した際に「Workers（高度なプログラム実行）」や「R2（ストレージ）」、「高度なセキュリティ設定」などの有料機能へスムーズに移行してもらうことを期待

**開発者の支持**:

- エンジニアが「Cloudflareは使いやすくて高性能だ」と認識すれば、所属する企業（大手企業など）でCloudflareの導入を提案するようになる

#### 3. インターネット全体のデータを収集・分析するため

- Cloudflareの使命: 「より良いインターネットを構築する」
- 多くのサイトが自社ネットワークを通ることで、**最新のサイバー攻撃の手法やトラフィックの傾向をリアルタイムで把握**できる
- この膨大なデータを分析することで、有料プランを契約している大企業（JALやShopifyなど）向けのセキュリティ精度をさらに高める

**営業時の説明ポイント**:

> 「怪しい無料サービスとは異なり、世界トップ企業の戦略的な投資によって成り立っている無料枠です。つまり、私たちは『Cloudflareが自社のインフラをより強固にし、将来の顧客を増やすための戦略』にうまく乗っかっている状態なのです。」
